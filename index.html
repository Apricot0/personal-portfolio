Machine Learning Related

  • Sentiment Analysis in Finance

    This project explores sentiment classification in financial sentences using various methods, including fine-tuning on small pre-trained models, zero-shot, and few-shot learning with Large Language Models (LLMs). Evaluation on financial news data reveals that the GPT-3.5 Turbo model, with six correctly classified samples, outperforms other approaches. These insights are crucial for practitioners and researchers in the field of sentiment analysis in finance.

  • Word2vec

    Implemented the word2vec algorithm in python and applied it to the text8 dataset. The word2vec algorithm is a two-layer neural network that processes text. Its input is a text corpus, and its output is a set of vectors: feature vectors for words in that corpus. While Word2vec is not a deep neural network, it turns text into a numerical form that deep neural networks can understand.

  • DAN/GRU

    Implemented a deep averaging network (DAN) and a gated recurrent unit (GRU) in python and applied them to the IMDB Sentiment dataset. The DAN is a simple neural network that averages the word vectors in a sentence to produce a sentence vector. The GRU is a recurrent neural network that processes a sentence one word at a time and produces a sentence vector. The sentence vector is then fed into a softmax classifier to predict the sentiment of the sentence.
